# 安装 APK

本页面介绍如何在安卓手机上安装 DesktopFriends 应用。

## 下载 APK

前往 [GitHub Releases](https://github.com/user/DesktopFriends/releases) 页面下载最新版本的 APK 文件。

- **Release 版本**：稳定版，推荐日常使用
- **Debug 版本**：调试版，包含更多日志信息

## 安装步骤

### 1. 允许安装未知来源应用

由于 APK 不是从 Google Play 下载的，需要先允许安装：

1. 打开手机 **设置**
2. 进入 **安全** 或 **隐私** 设置
3. 找到 **安装未知应用** 或 **未知来源**
4. 允许你的文件管理器或浏览器安装应用

::: warning 注意
不同手机品牌的设置路径可能略有不同
:::

### 2. 安装 APK

1. 使用文件管理器找到下载的 APK 文件
2. 点击 APK 文件
3. 按照提示完成安装

### 3. 授予权限

首次运行时，应用可能需要以下权限：

- **网络访问**：用于连接中继服务器
- **麦克风**（可选）：用于语音输入
- **存储**（可选）：用于加载自定义模型

## 常见问题

### 安装失败怎么办？

1. 确保手机存储空间充足
2. 检查是否已允许安装未知来源应用
3. 尝试卸载旧版本后重新安装

### 打开后闪退？

1. 确保安卓版本 >= 7.0
2. 尝试重启应用
3. 检查是否有足够的内存

### 找不到服务器？

1. 确保手机和电脑在同一局域网
2. 检查服务器是否正常运行
3. 尝试手动输入服务器 IP 地址
